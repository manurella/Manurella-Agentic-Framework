# Manurella frontend accessibility and visual QA fixture

Small static frontend fixture for evaluating a SaaS-style dashboard/settings panel. It uses only local HTML, CSS, and JavaScript.

## Files

- `index.html`
- `styles.css`
- `app.js`
- `server.js`

## Run locally

No dependency installation is required.

```sh
node server.js
```

Open the printed URL, usually:

```text
http://127.0.0.1:4173/
```

## Notes

This fixture intentionally presents realistic frontend behavior for evaluation. It does not include an issue answer key.
