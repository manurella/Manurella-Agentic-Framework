# Muse Benchmarks

## Purpose

These benchmark stubs define how Muse agents graduate from research candidates to accepted agents. They are not full datasets yet.

## Muse Lead Benchmarks

1. Phase routing: route ambiguous creative requests to the correct specialist with minimal clarification.
2. Scratchpad hygiene: update durable story state without storing transient draft noise.

Metrics:

- routing accuracy
- clarification turn count
- scratchpad precision
- author-intent preservation

## World Character Architect Benchmarks

1. Character web task: create contrasting character motivations and relationships from a premise.
2. World-rule task: build setting constraints that generate plot pressure without encyclopedic sprawl.

Metrics:

- internal consistency
- story utility
- canon/option separation

## Narrative Designer Benchmarks

1. Cliche subversion outline: produce a coherent outline that subverts a familiar premise without breaking causality.
2. Nonlinear structure task: include temporal shifts while preserving reader comprehension.

Metrics:

- structural coherence
- causal logic
- originality
- draftability

## Developmental Editor Benchmarks

1. Pacing diagnosis: identify sagging middle or rushed climax in an outline.
2. Motivation diagnosis: detect character action that lacks established cause.

Metrics:

- macro issue recall
- actionability
- author-intent preservation

## Line Style Editor Benchmarks

1. Voice preservation rewrite: improve a passage without changing facts or point of view.
2. Cliche reduction task: remove generic phrasing while preserving tone.

Metrics:

- semantic preservation
- voice preservation
- rhythm improvement
- cliche reduction

## Eventseed Subtasker Benchmarks

1. Outline-to-beats task: split a chapter outline into draftable beats.
2. Dependency task: preserve causal prerequisites across beat sequence.

Metrics:

- beat size
- dependency preservation
- no added plot facts

## Scene Drafter Benchmarks

1. Constrained beat expansion: expand one beat into prose without violating negative constraints.
2. Style-context task: draft with a compact style/context packet.

Metrics:

- beat adherence
- fluency
- elaboration quality
- constraint compliance

## Continuity Logic Checker Benchmarks

1. Synthetic contradiction set: detect timeline, object-state, and character-fact contradictions.
2. False-positive control: avoid flagging intentional ambiguity.

Metrics:

- contradiction recall
- false-positive rate
- evidence quality
- uncertainty calibration

## Copyeditor Benchmarks

1. Style-sheet enforcement: correct inconsistent capitalization, spelling, and terminology.
2. Voice-safe proofing: correct mechanics without flattening dialect or style.

Metrics:

- mechanical accuracy
- style-sheet adherence
- voice preservation
- false-positive corrections

## Context Coordinator Benchmarks

1. Relevant memory retrieval: return only prior events needed for a scene.
2. Summary compression: reduce a long chapter history to a compact packet without losing required facts.

Metrics:

- retrieval relevance
- compression ratio
- missing-critical-context rate
- token efficiency
